Déployer sur Vercel

Ajouter simplement ce dossier comme projet Vercel. Configurez les variables SQUID_RESEARCH_PATH et SQUID_COMMUNICATION_PATH pour pointer vers les dépôts locaux avant build.

Synchronisation automatique

Utilisez le script scripts/sync_content.py (à venir) ou lancez un `npm run sync` pour injecter les contenus actualisés avant chaque déploiement.
