Cette page isole les idées "bankable" issues de la boîte à idées. Aucune n'est présentée dans la landing tant qu'elle n'est pas en prod.

TalentSourcing / Pokédex Freelance

Reprise du MatchingEngine pour scorer les freelances, générer un rapport sourcé et l'envoyer automatiquement au client. Statut : concept validé, UX à prototyper.

Pipeline Malt + Slack Blocks

Réception des opportunités Malt dans Slack avec boutons interactifs (Accepter / Refuser / Plus d'infos / Reporter) basés sur Slack Bolt. Réponses pushées directement à Malt.

Sync auto Landing ↔ blog

Script `sync_content.py` pour agréger les contenus SquidCommunication et générer les articles/sections avant chaque build Vercel.

Mode "Pro" Freelance

Cockpit dédié (gestion offres, services, mails, scoring) afin d'exploiter les connecteurs existants dans un workflow orienté freelance.

Transparence

Chaque idée migrera vers la landing principale uniquement quand elle sera prête (code, tests, documentation).
